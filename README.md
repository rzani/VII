# VII - Vim, It is

Simple vimrc, here you find shortcuts and tweaks to improve your vim skills.

> Still in hard development

## How to install

```shell
git clone http://github.com/rzani/vii ~/.vii
sh ~/.vii/install.sh
```

### How to upgrade

```shell
cd ~/.vii
git pull --rebase
sh ~/.vii/install.sh
```
